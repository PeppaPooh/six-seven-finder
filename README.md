# Six Seven Finder
A Chrome extension to search for 67 meme related text in your active window. If any are found, it displays a bouncing-hands popup that gets more excited the more strings are found.

**Finds**: 
- `67` (but not inside a long stream of digits)
- `6.7`, `6,7`, `6-7`, `6–7`, `6—7`, `6 ~ 7`, `6 to 7`, `6 or 7`
- `six seven`, `sixty seven`, etc.
- …in English, Spanish, Portuguese, French, German, Russian, Turkish, Hindi, Chinese, Japanese, Korean, and Indonesian.


On Chrome Web Store
https://chromewebstore.google.com/detail/six-seven-finder/jkccolkaibjnakhkefclhjelebgafjge


## How to use
1. Install the extension.
2. Browse the web like normal.
3. When a “six seven” appears:
    - It gets highlighted.
    - A little popup appears in the bottom-right corner.
4. Click the popup card to cycle through matches.
5. Click Dismiss if you’ve had enough.
